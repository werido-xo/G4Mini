## 1. STM32G431 电机驱动核心板说明

由于本项目属于STM32的项目，设计上没有什么太多说明，主要是介绍一下功能和外设：

+ 下载方式：USB下载、SWD下载
+ 显示：一路8 Pin的 SPI接口（引脚间距0.5mm)
+ 多路PWM输出引脚
+ 一路UART调试接口
+ 一路USB通信接口
+ 一路电流检测接口
